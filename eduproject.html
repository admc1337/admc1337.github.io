Edustock Project

This project was a project assigned in my Structured Systems Analysis and Design class. The goal of the project was to go through the CI/CD process while following Agile methodology, specifically SCRUM method, in a team. We worked together as a team of five with different roles, I was the backend developer for our project. Every two weeks we did a sprint for a new phase of our project. We started with system requests, then requirements definition documents and use cases, then process modeling, then data modeling and starting design, ending with user interface design, program design, and system implementation.

Link to project: Edustock Github

My responsibitlites and contributions were the backend code and writing documentation for every phase. I was essentially "on-call" whenever a change wanted to be made for the prototype website and if the Docker images were not working properly. I handled the maintenance of the DockerHub and uploaded to the DockerHub whenever I made changes to the website. I also coded the HTML website and designed the website as well as prototyping how we would use SQL in combination with the website.
