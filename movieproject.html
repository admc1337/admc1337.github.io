Movie Project

This project was a hackathon project I came up with for my first hackathon ever. It was hackUMBC 2023 and I found a group of new coders, but the group fell apart so I ended up coding all of the backend and functionality. I had the idea of getting movie suggestions based on weather because in my friend group we like to watch movies, sometimes we just couldn't find a movie we felt fit the "vibe" for the moment. Getting a movie to fit the outside weather vibe would help get around this "vibe check".

Link to project: Movie Project Github

My idea for the project was to find weather API and make a call to that API to gather specific weather data and based on that weather data to web scrape imdb websites for movies in a specific genre and randomize a pick and return the pick to the user as the selected movie. I chose to write this in Python due to libraries, like BeautifulSoup, that allowed me to easily make API calls and webscrape websites. I also experimented with tkinter's library to try and make a simple app for the user to use and not rely on a terminal to execute commands. This project didn't win any awards at the hackathon, but I felt accomplished with my original idea.
