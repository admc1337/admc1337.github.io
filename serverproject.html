Home Server

This project was a personal at home project I decided to do. I graduated from college and had a lot of free time, so I wanted to start learning on my own some basic Linux and try to host things on my own server at home.

I built my server by using some PC parts I had left over from an old build and buying some new parts (PSU and M.2). After I made sure the computer turned on properly, I flashed Debian 12 distrubution onto a flashdrive and installed it onto the server. I decided to go with the desktop distro of Debian 12 just so I can familiarize myself with Linux as this was the first time I truly messed with it on my own. After setting it all up I installed xrdp and set up a firewall and fail2ban as my intention for the server was to port forward it through my mesh network and router so my friends could access it. I used the server to host Minecraft servers, and whenever my friends were having issues with the server I could remote into the server using xrdp or ssh and resolve any issues.
